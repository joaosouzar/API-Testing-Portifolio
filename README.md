# 📌 API Testing Portfolio - Postman + Newman

## 📋 Sobre o Projeto

Este repositório foi criado como um **portfólio de testes de API**, com o objetivo de demonstrar conhecimentos em testes de API e validação de serviços REST utilizando o Postman.

Os testes foram desenvolvidos utilizando a API pública **DummyJSON**, onde foram explorados diversos endpoints e realizadas diferentes tipos de requisições HTTP.

O objetivo principal deste projeto é simular um cenário real de QA, validando APIs através de:

- Requisições GET, POST, PUT e DELETE
- Validação de status code
- Validação de resposta (body)
- Geração de relatórios de execução com Newman

## ✅ Validações (Assertions)

Durante os testes foram aplicadas validações como:

- Status code da resposta
- Estrutura do JSON retornado
- Existência de campos obrigatórios
- Mensagens de erro em casos negativos
- Validação de dados retornados pela API

---

## Relatórios de Testes (Newman)

Os testes foram executados também via Newman, permitindo geração de relatórios automáticos.

Os relatórios incluem:

- Total de testes executados
- Testes aprovados
- Testes falhados
- Tempo de execução
- Detalhes das requisições
